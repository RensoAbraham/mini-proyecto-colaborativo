## Mini-Proyecto-Colaborativo

## Propósito

Aplicar Scrum y desarrollo colaborativo para construir un MVP fullstack funcional de Control Horario / Asistencia. Este proyecto utiliza Google Antigravity con agentes y Planning Mode, integrando una conexión real a base de datos mediante Supabase MCP.

El objetivo principal es entregar un producto mínimo viable que evidencie:
*   Una arquitectura bien definida.
*   Debugging efectivo.
*   Prácticas profesionales de trabajo en equipo.

##  Comenzando

Sigue estas instrucciones para obtener una copia del proyecto y ejecutarlo en tu máquina local.

### Prerrequisitos

Asegúrate de tener instalado lo siguiente:

*   *Node.js* 
*   *npm* 
*   *Git*

### Instalación y Ejecución

1.  *Clonar el repositorio* :
    bash
    git clone https://github.com/TU_USUARIO/Mini-Proyecto-Colaborativo.git
    

2.  *Navegar al directorio de la aplicación*:
    El código fuente del frontend se encuentra en la carpeta control-horario-app.
    bash
    cd control-horario-app
    

3.  *Instalar dependencias*:
    Ejecuta el siguiente comando para instalar todas las librerías necesarias.
    bash
    npm install
    

4.  *Ejecutar el servidor de desarrollo*:
    Inicia la aplicación en modo desarrollo.
    bash
    npm run dev
    

5.  *Abrir en el navegador*:
    La consola te mostrará el puerto en el que se está ejecutando (usualmente http://localhost:3000). Abre esa URL en tu navegador web.

## Funcionalidades Principales

*   *Login*: Acceso seguro a la plataforma.
*   *Registro*: Formulario para crear nuevas cuentas de usuario.
*   *Control de Horario*: Registro de entrada y salida
*   *Configuración*: Edita el perfil y gestiona la cuenta.
*   *Reportes*: Genera reportes de asistencia.
*   *Dashboard*: Vista principal con resumen de actividad.

## Tecnologías

*   *Frontend*: Next.js, React, Tailwind CSS.
*   *Herramientas AI*: Google Antigravity.
*   *Base de Datos*: Supabase.
